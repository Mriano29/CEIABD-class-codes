Función TTS local

Sustituye ElevenLabs por esto:

from TTS.api import TTS

# Cargar modelo una vez
tts = TTS(model_name="tts_models/es/css10/vits", progress_bar=False, gpu=False)

def local_tts(text):
    fname = f"static/{uuid.uuid4().hex}.wav"
    tts.tts_to_file(text=text, file_path=fname)
    return "/" + fname


Y en /send:

audio_url = local_tts(answer)
session["last_audio"] = audio_url
